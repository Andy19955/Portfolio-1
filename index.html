Projects

The Guinea Pigs

This is my exam project at Noroff Vocational School. The goal of this project was to demonstrate the skills learned over my first year of studies. I was tasked with creating a blog with a responsive design. The blog should contain at least a home page, an about page, a page with list of blog posts, a post specific page and a contact page. The contact form should have a form validation done in JavaScript. The blog's content should be stored on WordPress, installed as a headless CMS, and fetched via the WordPress REST API. The post specific page had to be able to dynamycally display all posts based on their post ID.

For this project I used HTML, CSS and JavaScript to create the Website. The site should consider WCAG principles and meta tags to improve SEO. GitHub is used for version control and the site is deployed to Netlify. The project also required to install Hotjar for user testing, and a report documenting the choices and process along the way.

RainyDays

This project has been used in multiple study modules at Noroff Vocational School, and has been developed more and more. The last module it was used in, Content Management Systems, it should end up as a responsive website displaying different products. The products should be stored in WordPress, installed as a headless CMS, with the WooCommerce plugin installed. By using the WooCommerce API, the product's data should be fetched with JavaScript. The page had to have at least 5 products stored in WooCommerce. The JavaScript should have the functinality to fetch a single product's data and render a grid of product data. There should be one product specific page that should have the ability to dynamically display all products based on their product ID.

For this project I used HTML, CSS and JavaScript to create the Website. The site should consider WCAG principles and meta tags to improve SEO. GitHub is used for version control and the site is deployed to Netlify.

Community Science Museum

This is my first semester project at Noroff Vocational School. The goal of this project was to demonstrate the skills learned over my first semester of studies. These skills included design principles, project management, communication technology and development with HTML/CSS. The target audience for this project was set to primary and middle school children (ages 7-15) and families with young children. The website should be responsive, informative, engaging and appealing to attract the target audience to visit the museum. For this project I was provided with media assets such as text and images.

For this project I used HTML and CSS to create the website. The site should consider WCAG principles and meta tags to improve SEO. GitHub is used for version control and the site is deployed to Netlify. The project also required to write a report documenting the choices and process along the way.
